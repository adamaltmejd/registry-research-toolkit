# Terms of Service

`microdata-tools-se` is early-stage research tooling. It is provided for local
testing and development workflows around Swedish administrative register data.

## Acceptable use

- Do not use the plugin to export row-level personal data from MONA.
- Use the plugin only in workflows that comply with the legal, contractual,
  and ethical rules that govern your data access.
- Review all generated scripts, prompts, and exported files before use.
