#### **Antal förvärvsarbetande på arbetsstället AntalSys**

AntalSys

Variabeln visar antal förvärvsarbetande på arbetsstället enligt RAMS.

Kvaliteten är avhängig avgränsningsmetoden för sysselsättning.

*I övrigt se Förvärvsarbetande.*

Vissa kategorier anställda kan inte kopplas till ett faktiskt arbetsställe (med CFAR-nummer som identitet). För dessa individer redovisas ingen uppgift (variabeln=0).
