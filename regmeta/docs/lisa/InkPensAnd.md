**Andel/grad för inkomstpension InkPensAnd**

InkPensAnd

(2003–)

Ger information om vilken grad av inkomstpension som tagits ut.

Kodförteckning:

00 = 0/0, 0 % utbetalt

01 = 1/1, 100 % utbetalt

02 = 1/2, 50 % utbetalt

03 = 3/4, 75 % utbetalt

04 = 1/4, 25 % utbetalt
